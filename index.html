Detect violence in video streams on the Edge

This project implements a 3D Convolutional Neural Network (3D-CNN) for detecting violent scenes in a video stream. The 3D-CNN is a deep supervised learning approach that learns spatiotemporal discriminant features from videos (sequence of image frames). The model is optimized for inferencing on the edge using Intel OpenVINO™ toolkit. Preliminary results have shown above-average accuracy.
